# 网页试玩说明

## 1. 构建网页版本

在 Cocos Creator 中操作：

1. 打开 `项目 -> 构建发布 (Build)`
2. 平台选择 `Web Desktop`
3. 点击 `构建`

构建完成后会生成目录：`build/web-desktop`

## 2. 本地启动网页游戏

在项目根目录执行：

```bash
npm run web:play
```

启动后浏览器访问：

```text
http://localhost:8080
```

如果 8080 被占用，可以指定端口：

```bash
PORT=9000 npm run web:play
```

PowerShell 下写法：

```powershell
$env:PORT=9000; npm run web:play
```

## 3. 手机访问

1. 手机和电脑连接同一个 Wi-Fi
2. 查看电脑局域网 IP（例如 `192.168.1.8`）
3. 在手机浏览器打开：

```text
http://192.168.1.8:8080
```

## 4. 常见问题

- 提示找不到 `build/web-desktop`：
  先在 Creator 里执行一次 Web Desktop 构建。
- 手机打不开：
  确认电脑防火墙允许 Node.js 端口 `8080`。
- 触屏操作不灵敏：
  手机浏览器优先使用 Chrome 或 Safari，避免微信内置浏览器。
